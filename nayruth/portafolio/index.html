Presentación alcance para proyecto

  • Servicio: Presentación para venta de producto

League of Legend For Beginners, Data-Dashboard

Esta página web que facilita el proceso de adaptación a millones de usuarios en categoría Beginner, para su introducción en el juego League of Legends. El proceso involucró entrevistas a usuarios, benchmark, prototipos de alta y baja fidelidad, definición del producto y User Persona, así como Lean Canvas.

Esta interfaz, con adaptaciones responsive y lineamientos de visual design encubre la manipulación de data estática con el proposito de ordenarla, filtrarla y adecuarla para hacerla amigable y accesible al usuario.

  • Cliente: Laboratoria
  • Fecha: Enero 2019
  • Servicio: App web - Data dashboard
  • Github: Repositorio del proyecto

Burger Queen WebApp

¿Alguna vez te has cuestionado si tus llamadas y mensajes quedan registrados en alguna base de datos en tu compañia telefónica? ¿O si existen registros diarios de tus ubicaciones por tener activo el Wi-Fi?, esta era digital trae consigo dos caras de la moneda, y una de ellas, es la privacidad de nuestra información. De allí nace la idea, como parte de un programa completo que nos permita entender la importancia de proteger nuestra información ante las grandes empresas y corporaciones.

Se plantea que esta interfaz sea parte de un Taller Interactivo dictado a escuelas, universidades y corporaciones, donde los participantes puedan cifrar y descrifrar palabras con el método de Cifrado Cesar, dando así sus primeros pasos ante esta problematica mundial.

  • Cliente: Laboratoria
  • Fecha: Julio 2019
  • Servicio: WebApp
  • Github: Repositorio del proyecto

Librería md-links003

¿A quien no le ha pasado encontrarse con un link roto o que no exite en los famosos readme de Github ?, md-links003 verifica, válida y realiza una estadística de cualquier archivo que utilize el lenguaje markdown

Esta libreria esta publicada en npm para que otros desarrolladores puedan utilizarlo, se puede utilizar como una API (Application Programming Interface) o CLI (Command-line Interface).
Tecnologías utilizadas: Node.js (Asíncrono), Vanilla JS.

  • Cliente: Laboratoria
  • Fecha: Junio 2019
  • Servicio: Librería md-links
  • Github: Repositorio del proyecto

Bésame Mucho, Red Social

Bésame Mucho, red social creada para padres de familia que quieran promover la educación respetuosa para sus hijos. Cuenta con funciones básicas de una red social: publicar, editar, eliminar y configurar privacidad de posts, así como editar mi perfil, iniciar sesión con Google, Facebook y correo electrónico.

Tecnologías utilizadas: Firebase (manipulación de data obtenida desde la UI), Vanilla JS ( Vanilla JS para la implementación de todo el proyecto, convirtiendola en una SPA (Single Page Application) responsive).

  • Cliente: Laboratoria
  • Fecha: Mayo 2019
  • Servicio: Red social
  • Github: Repositorio del proyecto
